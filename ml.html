Check it Out!

Interested in contributing to JEMMA ? The current JEMMA source code can be accessed on the official JEMMA GitHub page. You can also download the latest snapshot of the full code repository in .zip or .tar.gz format.

The Revision History of the project is hosted on the JEMMA Wiki, together with other developer-oriented documentation. Known Bugs, TODOs and information about plans for future releases is tracked in the project tracker (organized by milestone).

Stay Tuned!

The main announcements about JEMMA are normally published on the Developers Blog, toghether with interesting project-related news.

Get in contact!

The preferred way to contact the JEMMA development team and other users/developers is to join the JEMMA general mailing list (click here to subscribe).
