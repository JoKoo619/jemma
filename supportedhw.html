Icon Legend

Here the meaning of icons on the "status" level.

  • = Verified (with first JEMMA version where verification has been completed)
  • = Should work (= compliant with supported ZigBee standard or already tested with some partial component of JEMMA, but not tested with the full solution)
  • = Incompatible (proven not to work - with link to the related issue on the JEMMA Tracker)

Note: text in the tables of this page has been provided either by vendors or by contributors which have integrated/tested HW with JEMMA.
